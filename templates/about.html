Version 2.0

The first version of my Squirrel Detector was a very simple project that ran on a Raspberry Pi. It used RoboFlow for training the model and for local inference. It was a great project, but the training wasn't very accurate and I wanted to be able to use local images for training.

NVIDIA's Jetson Nano seemed like a great platform to use for an updated version. I had several goals in mind as I started designing the new version:

  • Easy to use and self-contained
  • Can collect images and train locally
  • Provide an educational experience
  • Meet requirements for NVIDIA Jetson AI Specialist certification project

You will notice that I have departed from the typical approach to annotation. Usually you would have a tool that allows you to draw a bounding box around the object of interest. This would have been more complicated to implement, so I decided to use a simpler approach.

When an image is captured, the code highlights the area in the image where motion was detected. The coordinates for that area are saved with the image, and will be used as the suggested annotation section for the image. Instead of manually drawing boxes, the user can just select the images that have a squirrel and a bounding box that is in the right place.
